OpenASP AX チャットサービス マニュアル

## 概要
Gemmaモデルを使用したAIアシスタントチャットサービスです。

## 主要機能
1. テキストチャット: 自然言語での質問と回答
2. ファイルアップロード: 画像や文書ファイルの添付
3. RAG機能: 関連文書からの情報検索と活用
4. マルチモーダル: 画像分析とテキスト処理

## 使用方法
1. チャット画面でメッセージを入力
2. Enterキーで送信（Shift+Enterで改行）
3. ファイルアップロードボタンでファイル添付
4. AIアシスタントが自動的に応答

## サポートファイル形式
- 画像: JPG, PNG, GIF, WebP
- 文書: PDF, DOC, DOCX, TXT, MD
- その他: プロジェクト関連ファイル

## RAG機能
- public/RAGディレクトリの文書を自動検索
- 関連情報を含めた回答生成
- 参考資料の表示

## 技術仕様
- AI モデル: Gemma 2B
- バックエンド: Python Flask
- API ポート: 3006
- Ollama ポート: 3014
